# Contract Form Demo

This is a demo page for Contract Form using react-hook-form and jspdf

### Install

```bash
yarn
```

Serve with hot reload at <http://localhost:5173>.

```bash
yarn run dev
```

### Lint

```bash
yarn run lint
```

### Typecheck

```bash
yarn run typecheck
```

### Build

```bash
yarn run build
```

### Test

```bash
yarn run test
```

View and interact with your tests via UI.

```bash
yarn run test:ui
```
